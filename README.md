# Unigate - Video Surveillance Dashboard

Una dashboard per il monitoraggio video con supporto per HLS streaming e visualizzazione multi-camera.

## Caratteristiche

- Dashboard responsive per il monitoraggio di multiple telecamere
- Supporto per streaming HLS
- Interfaccia moderna con tema scuro
- Visualizzazione in tempo reale dello stato dei feed video

## File

- `webcam-dashboard.html` - Dashboard principale con interfaccia di controllo

## Utilizzo

Aprire `webcam-dashboard.html` in un browser web moderno.
